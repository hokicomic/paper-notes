前沿 AI 研究與論文追蹤導航｜更新版

用於快速抓取最新 AI 論文、判斷是否值得深讀、追蹤程式碼與資料集，並把高價值論文交給本專案做「價值初篩 → 新知萃取 → 雙欄批注 HTML」。

更新日期：2026-07-05 用途：AI / ML / Agent / RAG / Multimodal / Benchmark 所有網站按鈕皆為可點擊連結
使用原則：不要把所有來源都當成同等重要。每日只看「趨勢入口」，每週用「引文地圖」擴展主題，高價值論文才丟進本專案做完整 HTML 批注。低價值論文只做速讀卡。

1. 最新論文與社群熱度入口

適合每天 5–10 分鐘掃描。核心任務是找候選論文，不是立刻精讀。

每日必看

社群投票、Daily / Weekly / Monthly
造訪網站 →

AI 社群熱門論文入口，可用 Daily、Weekly、Monthly 篩選，也常連到 arXiv、GitHub、Demo、模型或資料集。

最適合：每天挑 3–5 篇候選論文做價值初篩。
趨勢補充
Hugging Face Trending Papers
熱門論文與社群互動
造訪網站 →

適合看近期被社群大量討論的 AI 論文。若只想快速掌握主線，優先看這裡。

最適合：抓熱門題材、agent、LLM、multimodal 等方向。
討論脈絡
alphaXiv

可探索 trending papers、近期活動、作者追蹤與論文討論。適合看一篇論文是否真的引起研究社群反應。

最適合：判斷論文熱度是否只是標題黨，或有實質討論。
個人化篩選
arXiv Sanity Light
arXiv 熱門與個人化推薦
造訪網站 →

適合快速看近期 arXiv 熱門論文與建立個人化篩選習慣。若服務不穩，請改用 Hugging Face Papers 與 arXiv 官方入口。

最適合：作為每日掃描的第二來源。

3. 程式碼、資料集、Benchmark 與可復現性

如果你的目標是學到可遷移的新方法，這一類比純摘要更重要。

可復現

論文、程式碼、資料集、排行榜
造訪網站 →

用來確認論文是否有 code、dataset、method 與 leaderboard。若只想讀能實作的論文，這是重要篩選器。

最適合：可操作性評分與復現任務設計。
程式碼搜尋
CatalyzeX
幫論文找開源實作
造訪網站 →

可搜尋 AI 與其他研究論文的開源程式碼，也有瀏覽器擴充功能，瀏覽 arXiv、Scholar、GitHub 時能提示相關 code。

最適合：快速判斷一篇論文能不能實作。
模型/資料
Hugging Face Models

造訪網站 →

讀模型論文時，用來確認是否有實際模型權重、推論範例、license 與社群使用情況。

最適合：模型可用性與產品應用評估。
資料集

造訪網站 →

用來檢查資料來源、欄位、license、偏差、限制與可否直接載入。資料集論文尤其需要看這裡。

最適合：資料品質與可遷移性分析。
