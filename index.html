7. 建議閱讀工作流

避免資訊過載。這個頁面不是要你每個網站都看，而是把不同任務交給不同來源。

每日 10 分鐘

  1. 先看 Hugging Face Daily / Trending，抓 3–5 篇候選論文。
  2. 若想補中國研究動向，再看 ChinaRxiv AI / CS 或 ChinaXiv 計算機科學。
  3. 只記錄題名、URL、是否有 code / model / dataset，不要立即深讀。

每週 40 分鐘

  1. 把候選論文丟到 Semantic Scholar / Litmaps，看前置論文、後續引用與相近工作。
  2. 用 Papers with Code、Hugging Face、GitHub、OpenI 檢查可復現性。
  3. 只把價值初篩 18 分以上的論文交給本專案做完整雙欄批注 HTML。
問題優先使用來源判斷重點
今天 AI 社群在看什麼？Hugging Face Papers、alphaXiv熱度、討論、是否有程式碼或 demo
這篇論文是否真的進入學術脈絡？Semantic Scholar、Litmaps、OpenReview引用、相關工作、審稿質疑
有沒有中國研究團隊或中文預印本？ChinaRxiv、ChinaXiv、AMiner、PubScholar是否為預印本、是否有英文版、是否有正式發表版本
能不能實作？Papers with Code、CatalyzeX、Hugging Face、GitHub、OpenIcode、dataset、model、license、維護狀態
大廠最近真正押什麼方向？OpenAI、DeepMind、Meta、Anthropic、Microsoft、NVIDIA模型報告、system card、安全文件、基礎設施研究

可直接貼給本專案的論文整理 prompt
